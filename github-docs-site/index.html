Github запустил сайт по продуктам - docs.github.com.

Сергей Мочалов 02/07/2020

1 июля 2020 года Github запустил новый сайт с документацией по его продуктам.

Адрес сайта - https://docs.github.com/

Этот новый сайт вобрал в себя контент, который был на help.github.com и developer.github.com. С помощью docs.github.com, вы сможете найти всю документацию по продуктам Github в одном месте. 

В будущем сайт получит несколько языковых версий, на данный момент поддерживается японский, упрощенный китайский, бразильский португальский и испанский языки.

Запуск проекта docs.github.com это первый шаг в создании единого портала, где вы найдете информацию о новых инструментах и подходах. Мы продолжим совершенствовать этот опыт, чтобы у вас были все инструменты, необходимые для успешного использования GitHub.
